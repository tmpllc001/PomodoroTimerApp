# 🎯 Phase 4 実際の使い方ガイド

## 📋 統計機能の実際のタブ構成

アプリを起動すると、以下のタブが表示されます：

### 1. **統計タブ** (基本統計)
- **内容**: シンプルなテキスト表示
- **機能**: 基本的な統計情報のみ
- **インタラクティブ機能**: なし

### 2. **分析・可視化タブ** (Phase 4 高度機能)
#### 利用可能な機能:
- **📊 テンプレートレポート選択**
  - ドロップダウンメニューから選択:
    - `daily_summary` - 日次サマリー
    - `weekly_analysis` - 週次分析  
    - `comprehensive` - 包括的レポート

- **📈 可視化機能**
  - 生産性タイムライン表示ボタン
  - フォーカスヒートマップ表示ボタン
  - 中断分析表示ボタン
  - セッション性能分析ボタン

- **📊 比較分析**
  - 平日 vs 週末比較ボタン
  - 時間帯別比較ボタン

### 3. **AI予測・エクスポートタブ** (Phase 4 高度機能)
#### 利用可能な機能:
- **🤖 AI予測機能**
  - フォーカス予測ボタン
  - 最適作業時間予測ボタン
  - 生産性トレンド予測ボタン
  - モデル再トレーニングボタン

- **📄 エクスポート機能**
  - PDF包括レポート生成ボタン
  - Excel詳細データ生成ボタン

- **⏰ 自動レポート設定**
  - 日次レポート生成ボタン
  - 週次レポート生成ボタン
  - 月次レポート生成ボタン

## 🎯 実際の操作手順

### Step 1: アプリを起動
```bash
cd /mnt/d/00_tmpllc/12_pomodoro/PomodoroTimerApp
python pomodoro_phase3_final_integrated_simple_break.py
```

### Step 2: 高度統計機能を利用
1. **基本統計を確認**: 「統計」タブをクリック
2. **高度分析を利用**: 「分析・可視化」タブをクリック
3. **AI予測を利用**: 「AI予測・エクスポート」タブをクリック

### Step 3: 具体的な機能の使い方

#### 📊 分析・可視化タブでの操作:
1. **テンプレートレポート**: ドロップダウンから選択して「生成」
2. **グラフ表示**: 各ボタンをクリックしてグラフを表示
3. **比較分析**: 比較ボタンをクリックして分析結果を表示

#### 🤖 AI予測・エクスポートタブでの操作:
1. **予測実行**: 予測ボタンをクリックして結果を確認
2. **レポート生成**: PDF/Excelボタンでレポートを生成
3. **自動設定**: 自動レポートボタンでスケジュール設定

## 📁 生成されるファイル

### レポートファイル
- `data/reports/comprehensive_report_YYYYMMDD.pdf`
- `data/reports/detailed_data_YYYYMMDD.xlsx`
- `data/reports/daily_report_YYYYMMDD.json`

### 画像ファイル
- `data/charts/productivity_timeline_YYYYMMDD.png`
- `data/charts/focus_heatmap_YYYYMMDD.png`

## ⚠️ 注意事項

1. **初回利用時**: 十分なセッションデータが必要
   - 最低5セッション以上推奨
   - テストデータは既に30セッション生成済み

2. **AI予測機能**: 
   - scikit-learnがない場合は基本予測のみ
   - 精度向上にはデータ蓄積が必要

3. **エクスポート機能**:
   - reportlab/openpyxlがない場合はJSON形式
   - フォールバック機能により基本機能は利用可能

## 🎉 実際に試してみよう！

1. アプリを起動
2. 「分析・可視化」タブをクリック
3. ドロップダウンから「comprehensive」を選択
4. 「生成」ボタンをクリック
5. 生成された包括的レポートを確認！
